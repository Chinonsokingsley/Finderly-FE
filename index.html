Welcome to the Finderly documentation page! Here's how you can use the tool:

1. Home Page

  • Select a Country from the dropdown.
  • Choose a Theme (e.g., Human, Animal, Building).
  • Enter an Extension (if applicable).
  • Click Submit to proceed.

2. Extension Request

  • Indicate if you have an existing SPARQL query.
  • Select a Theme (e.g., Human, Organization).
  • Provide a Title for your request.
  • Paste your SPARQL query or describe your extension.
  • Enter a Reason for the request.
  • Provide your Wikimedia Username.
  • Click Submit to send your request.

3. Search Results

  • View available and non-available articles/items.
  • Switch between languages using the Select Language dropdown.
  • Choose a format (e.g., Plain Text, JSON, CSV) to copy/save results.

4. Extension Tracker

  • Track the status of your extension requests.
  • View the Title, Theme, Reason, Username, and Status of each request.

If you have any questions, feel free to reach out to our support team.
